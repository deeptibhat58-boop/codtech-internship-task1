This project demonstrates a

============================
ARCHITECTURE DESIGN
============================

CLOUD 1 - GitHub:
- Service: Cloud Storage
- Repository: codtech-internship-task1
- Access: Public
- Files: hello.txt, info.txt
- URL: github.com/deeptibhat58-boop/
  codtech-internship-task1

CLOUD 2 - UptimeRobot:
- Service: Cloud Monitoring
- Monitor: GitHub Repository
- Uptime: 100%
- Alerts: Email notifications
- Dashboard: Active metrics

============================
INTEROPERABILITY
============================
Both platforms work together:

1. GitHub stores the files
   (Cloud Storage Service)

2. UptimeRobot monitors GitHub
   (Cloud Monitoring Service)

3. If GitHub goes down:
   - UptimeRobot detects it
   - Sends email alert
   - Shows on dashboard

This shows INTEROPERABILITY
between two cloud platforms!
